## Installation

```bash
# Clone this project
$ git clone https://github.com/andrefangeloni/wallet-test.git

# Access
$ cd wallet-test

# Install dependencies
$ yarn 

# Start server
$ json-server --watch db.json

# The server will initialize in the <http://localhost:3000>

# Run Android
$ yarn android

# Run iOS
$ yarn pods && yarn ios
```
